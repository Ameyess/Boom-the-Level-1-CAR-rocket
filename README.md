# Boom-the-Level-1-CAR-rocket
This is Boom (my level 1 certification rocket)

Boom is a single-stage Level 1 high-power rocket designed and built from scratch for CAR L1 certification. It features a 3D-printed nose cone, CNC-machined trapezoidal fins and a PerfectFlight Firefly altimeter for apogee detection as well as ejection charge deployment.

# Why I Built This
I've been fascinated by space and rockets since I was 10... playing games like Universe Sandbox and dreaming about what's out there beyond Earth. I loved learning how propulsion systems and aerodynamics work, seeing how to make things glide more smoothly through the air and how air can be a little funky. Since I was 10, I've been keeping up with this YouTuber called Integza and he really brought me into the world of rockets and propulsion. On top of all this I want to pursue a career in aerospace engineering. Instead of building a wind tunnel or a similar project, I wanted to get my L1 certification. I wanted to actually build something real before entering the field. Boom is that project: a fully designed, simulated, and flown (hopefully) rocket that I can point to and say I made it.

# Simulated Flight Performance 
To perform the simulations, I used OpenRocket. I first designed the whole rocket in OpenRocket then moved it to CAD for it to be fully 3D modelled and ready to construct. Simulations were run using the HP-H195NT-10 motor.

| Parameter | Value |
| -------- | -------- |
| Projected Apogee | 1484m | 
| Max Velocity | 470 m/s |
| Stability Margin| 1.8 Cal |
|Recovery | Parachute deployed at apogee|

# Design Overview
## Airframe
- Nose cone: 3D printed in PLA and includes an integrated altimeter bay in the shoulder
- Body tube: 18in long with a diameter of 1.325in
- Fins: Trapezoidal profile, CNC-machined from polycarbonate and chamfered to a knife's edge

## Recovery System
- Single parachute ejected at apogee
- Ejection charge triggered by PerfectFlight Firefly altimeter

## Avionics
- Altimeter: PerfectFlight Firefly: handles apogee detection and ejection charge deployment
- Mounted inside a dedicated compartment in the nose cone shoulder

## Motor
- Motor: Aerotech H135W-14A (or HP-H195NT-10 for simulation)
- Motor retained with a retaining ring (3d Printed)
- Rail lugs for launch rail guidance
